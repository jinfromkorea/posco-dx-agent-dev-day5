# 주간 보고서 작성 스킬

## 사용 시기
- 사용자가 주간 보고서, 업무 보고, 주간 리포트 작성을 요청할 때

## 보고서 양식 (반드시 아래 형식을 따를 것)

### [주간 업무 보고서]
- **보고 기간**: YYYY.MM.DD ~ YYYY.MM.DD
- **작성자**: (사용자 이름 또는 '작성자')

#### 1. 금주 실적
- 완료된 업무를 불릿 포인트로 정리

#### 2. 주요 이슈
- 발생한 문제점과 해결 방안

#### 3. 차주 계획
- 다음 주 예정 업무

#### 4. 건의사항
- 지원 요청 또는 건의

## 규칙
- 각 섹션은 반드시 포함할 것
- 불릿 포인트로 간결하게 작성
- 구체적 날짜와 수치를 포함
