# 시스템 접속 안내 스킬

## 사용 시기
- 사용자가 특정 시스템의 접속 방법을 물어볼 때
- SSO 연동 여부 확인 요청 시
- 비밀번호 변경, 계정 신청 방법 안내 시
- 신규 입사자 온보딩 시스템 안내 시

## 답변 양식 (반드시 아래 형식을 따를 것)

### 🖥️ [시스템명] 접속 안내

| 항목 | 내용 |
|------|------|
| URL | [접속 주소] |
| SSO | ✅ 연동 / ❌ 미연동 |
| 담당 부서 | [부서명] |
| 접근 범위 | [사내 / 사내·사외] |

#### 접속 방법
[단계별 접속 방법을 번호 목록으로 안내]

#### 주의사항
- SSO 미연동 시: 별도 계정 발급 절차 안내
- 사외 접속 필요 시: VDI 이용 안내

---

## 규칙
- get_system_detail 도구를 항상 먼저 호출하여 최신 정보를 가져올 것
- 테이블과 번호 목록으로 명확하게 구조화할 것
- SSO 연동 여부를 가장 먼저 강조할 것
- 담당 부서 연락처가 있으면 반드시 포함할 것
- 사외 접속이 불가능한 시스템은 VDI 사용을 안내할 것
